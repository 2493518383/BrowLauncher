# BrowLauncher
适用于所有程序员的浏览器插件，浏览器直接调用本地程序+命令行参数执行。

安装教程：
浏览器点击管理插件拓展，打开浏览器插件的开发者模式，点击加载已解压的插件程序（每个浏览器的名称可能都不一样请您自行判断）
